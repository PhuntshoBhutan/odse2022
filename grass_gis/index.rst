Introduction

Software used in this training materials
========================================

.. figure:: ./images/grass-gis-logo.png
   :class: small
          
* `GRASS GIS <http://grass.osgeo.org>`__ **version 8** for geospatial
  raster, imagery, and vector data processing and providing spatial
  analysis.
* `sentinelsat library <https://pypi.python.org/pypi/sentinelsat>`__
  utility to search and download Copernicus Sentinel satellite images.

Sample data
===========

For purpose of this training materials a `sample data including GRASS
8 binaries
<https://geo.fsv.cvut.cz/geoharmonizer/odse_workshop_2022/grass-gis-data.zip>`__ (690MB)
has been created based on Open Data covering ODSE pilot tiles.
